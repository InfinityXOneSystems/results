# Proof of Truth Collection Procedures

## Overview

This document outlines the procedures for collecting, documenting, and archiving proof that validates every claim made about a property or business opportunity.

---

## Proof Categories

### 1. Address Proof

**Objective:** Prove the address is real, valid, and matches the property

**Collection Methods:**

**Method 1A: USPS Verification**
```
1. Visit: https://tools.usps.com/zip-code-lookup.htm
2. Enter address
3. Verify address is recognized by USPS
4. Screenshot result showing:
   - Address as entered
   - USPS standardized address
   - ZIP code
   - Delivery confirmation
5. Save as: address_usps_verification_[YYYYMMDD].png
6. Record: URL, timestamp, result
```

**Method 1B: Google Maps Verification**
```
1. Visit: https://maps.google.com
2. Enter property address
3. Verify property appears on map
4. Screenshot showing:
   - Address in search box
   - Property location on map
   - Coordinates
   - Street view (if available)
5. Save as: address_google_maps_[YYYYMMDD].png
6. Record: URL, timestamp, coordinates
```

**Method 1C: County Assessor Verification**
```
1. Visit: County Property Appraiser website
2. Search for property by address
3. Verify property record exists
4. Screenshot showing:
   - Property address
   - Parcel number
   - Property details
   - Assessed value
5. Save as: address_county_assessor_[YYYYMMDD].png
6. Record: URL, timestamp, parcel number
```

**Proof Package:**
- USPS verification screenshot
- Google Maps screenshot
- County assessor record
- All with timestamps and URLs

### 2. Ownership Proof

**Objective:** Prove current property owner and ownership chain

**Collection Methods:**

**Method 2A: County Deed Records**
```
1. Visit: County Clerk/Recorder website
2. Search deed records by address or parcel number
3. Find most recent deed
4. Download or screenshot deed showing:
   - Grantor (previous owner)
   - Grantee (current owner)
   - Property description
   - Deed date
   - Recording date
   - Deed type (Warranty, Quitclaim, etc.)
5. Save as: deed_[owner_name]_[YYYYMMDD].pdf
6. Record: Recording number, date, URL
```

**Method 2B: Tax Assessor Owner Information**
```
1. Visit: County Tax Assessor website
2. Search property by address
3. Screenshot property record showing:
   - Current owner name
   - Owner mailing address
   - Ownership type (individual, corporate, etc.)
   - Tax ID
   - Parcel number
4. Save as: tax_assessor_owner_[YYYYMMDD].png
5. Record: URL, timestamp, tax ID
```

**Method 2C: Property Appraiser Record**
```
1. Visit: County Property Appraiser website
2. Search property by address
3. Screenshot property card showing:
   - Owner name
   - Mailing address
   - Property details
   - Assessed value
   - Last update date
4. Save as: property_appraiser_owner_[YYYYMMDD].png
5. Record: URL, timestamp, last update date
```

**Proof Package:**
- Deed record (PDF)
- Tax assessor owner record (screenshot)
- Property appraiser record (screenshot)
- All with timestamps and URLs

### 3. Distress Proof

#### 3A: Foreclosure Proof

**Objective:** Prove property is in foreclosure

**Collection Methods:**

**Method 3A-1: HUD Foreclosure Database**
```
1. Visit: https://www.hud.gov/foreclosure
2. Search for property by address
3. If found, screenshot showing:
   - Property address
   - Foreclosure status
   - Case number
   - Foreclosure date
   - Lender information
4. Save as: foreclosure_hud_[YYYYMMDD].png
5. Record: URL, timestamp, case number
```

**Method 3A-2: County Clerk Records**
```
1. Visit: County Clerk website
2. Search court records by address
3. Find foreclosure case
4. Download Notice of Default/Foreclosure showing:
   - Property address
   - Case number
   - Plaintiff (lender)
   - Defendant (owner)
   - Filing date
   - Foreclosure timeline
5. Save as: foreclosure_notice_[case_number]_[YYYYMMDD].pdf
6. Record: Case number, filing date, URL
```

**Method 3A-3: Auction Listing**
```
1. Visit: Auction.com or similar site
2. Search for property
3. If listed, screenshot showing:
   - Property address
   - Auction date
   - Starting bid
   - Property description
   - Auction status
4. Save as: foreclosure_auction_[YYYYMMDD].png
5. Record: URL, timestamp, auction date
```

**Proof Package:**
- HUD foreclosure listing (screenshot)
- Notice of Default/Foreclosure (PDF)
- Auction listing (screenshot)
- All with timestamps and URLs

## Verification Checklist

For each property, verify:

- [ ] Address verified by USPS
- [ ] Address verified by Google Maps
- [ ] Address verified by county assessor
- [ ] Ownership verified in deed records
- [ ] Ownership verified in tax records
- [ ] Ownership verified in appraiser records
- [ ] Distress type confirmed
- [ ] Distress documents collected
- [ ] Price/value verified
- [ ] All documents timestamped
- [ ] All documents sourced
- [ ] Metadata file created
- [ ] All files organized
- [ ] Validation score calculated
- [ ] Final approval given

---

## Quality Assurance

### Document Quality Standards

- All screenshots must be clear and readable
- All PDFs must be complete and legible
- All timestamps must be accurate
- All URLs must be recorded
- All sources must be verified

### Audit Trail

- Record who collected each document
- Record when each document was collected
- Record how each document was collected
- Record any issues or exceptions
- Maintain complete audit trail
